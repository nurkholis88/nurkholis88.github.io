Hapus Data Ganda pada LibreOffice Calc

Beberapa waktu yang lalu, ada yang bertanya terkait cara hapus data duplikat / ganda pada LibreOffice Calc. Selidik punya selidik, ternyata fitur ini tidak tersedia secara native di LibreOffice. Namun, bukan berarti tidak ada cara menghapus data ganda tersebut.

Pada kesempatan kali ini, kita akan bahas beberapa cara yang dapat kita lakukan untuk menangani data ganda / duplikat pada LibreOffice Calc.

Ekstensi Remove Duplicate

Meski fitur Remove Duplicate (hapus data ganda) belum ada secara native pada LibreOffice, tetapi ada ekstensi yang dapat kita pasang untuk melakukan hal tersebut.Kita dapat mengunduhnya pada tautan Remove Duplicates atau Remove Duplicates Fast. Keduanya mempunyai fitur dan cara kerja yang sama dengan fitur Remove Duplicates pada Ms. Excel.

Instalasi Ekstensi

Untuk menginstal Ekstensi Remove Duplicates / Remove Duplicates Fast, langkah-langkahnya adalah sebagai berikut:

  1. Klik Meno Tools > Extension Manager
  2. Klik Add, kemudian pilih File ekstensi (format .oxt) yang akan dipasang
  3. Jika muncul pesan Term & Agreement, scroll ke bawah kemudian klik Accept
  4. Tunggu proses hingga selesai, kemudian restart LibreOffice

Penggunaan

Untuk menggunakan Ekstensi ini, langkah-langkahnya:

  1. Blok Range / Tabel yang hendak dihapus datanya
  2. Klik menu Data > Remove Duplicates
  3. Tekan Ctrl+Click untuk memilih / batal memilih kolom-kolom yang dianalisa data duplikatnya
  4. Klik OK

Cara Alternatif

Selain menggunakan ekstensi, ada beberapa alternatif lain menghapus data duplikat pada LibreOffice Calc. Sebenarnya, ini tidak benar-benar menghapus data duplika, tetapi lebih tepat disebut sebagai memilih data unik dari data ganda yang sudah ada.

Menggunakan Filter

Dengan menggunakan Standar Filter, kita bisa memilih data unik dan menyalinnya ke Sheet / Range lain. Caranya:

  1. Blok Range / Tabel yang hendak difilter
  2. Klik Menu Data > More Filter > Standar Filter
  3. Atur kolom-kolom yang hendak difilter
  4. Klik Option, centang No Duplications, dan Copy results to:
  5. Atur lokasi untuk meletakkan hasil Filter, Misal ke Sheet2$A$1
  6. Klik OK

Demikian tadi beberapa cara menghapus data ganda pada LibreOffice Calc. Semoga kedepannya fitur Remove Duplicates ini bisa ada secara native pada LibreOffice Calc.
